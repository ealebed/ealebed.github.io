Установка Percona Server 5.6 в Debian Wheezy

Jan 7, 2016 17:24 · 137 words · 1 minute read percona debian

Скачать исходники, tar-архив, .rpm или .deb-пакеты Percona Server можно с официального сайта, но для тех, кто любит устанавливать ПО через менеджер пакетов, рассмотрим вариант установки Percona Server версии 5.6 из официального репозитория в ОС Debian Wheezy. Давайте разберемся!

Для начала необходимо скачать .deb пакет с веб-сайта:

wget https://repo.percona.com/apt/percona-release_0.1-3.$(lsb_release -sc)_all.deb

В зависимости от кодового имени вашей операционной системы будет скачан пакет, который необходимо установить с помощью пакетного менеджера dpkg:

dpkg -i percona-release_0.1-3.$(lsb_release -sc)_all.deb

Примечание. При установке будут добавлены официальные репозитории от Percona, убедиться в этом можно проверив файл /etc/apt/sources.list.d/percona-release.list.

Обновим информацию о пакетах в системе:

aptitude update

Установка Percona Server 5.6 в Debian Wheezy выполняется командой:

aptitude install percona-server-server-5.6

Проверим состояние пакета:

dpkg -l | grep percona-server-server
ii  percona-server-server-5.6             5.6.16-64.2-569.wheezy        amd64        Percona Server database server binaries

На этом все, установка Percona Server в операционной системе Debian Wheezy завершена.
