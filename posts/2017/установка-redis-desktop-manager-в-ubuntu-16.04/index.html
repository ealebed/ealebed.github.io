Установка Redis Desktop Manager в Ubuntu 16.04

Dec 21, 2017 11:40 · 208 words · 1 minute read redis

Redis Desktop Manager (RDM) — приложение для управления базами данных Redis с открытым исходным кодом для Windows, Linux и MacOS. RDM поддерживает шифрование SSL/TLS, SSH-туннели и облачные экземпляры Redis, такие как Amazon ElastiCache, Microsoft Azure Redis Cache и Redis Labs.

Давайте разберемся с установкой Redis Desktop Manager в ОС Ubuntu 16.04!

RDM предлагает простой в использовании графический интерфейс для доступа к Redis DB и выполнения некоторых основных операций (просмотру/удалению/изменению ключей, выполнению команд через оболочку).

В официальной инструкции по установке сказано, что для загрузки deb-пакета RDM требуется подписка ($1.99 per month) и есть возможность собрать данную утилиту из исходников.

Скачиваем исходники:

git clone --recursive https://github.com/uglide/RedisDesktopManager.git -b 0.9 rdm && cd ./rdm

cd src/ && ./configure

source /opt/qt59/bin/qt59-env.sh && qmake && make 

make install

cd /usr/share/redis-desktop-manager/bin && mv qt.conf qt.backup

Запустить Redis Desktop Manager можно из консоли с помощью команды:

./usr/share/redis-desktop-manager/bin/rdm

Все работает, но, как оказалось, есть гораздо более простой вариант установки Redis Desktop Manager в ОС Ubuntu 16.04.

На этой странице смотрим актуальную версию RDM с готовым deb-файлом (на момент написания статьи это 0.9.0-alpha4).

Скачиваем пакет:

wget https://github.com/uglide/RedisDesktopManager/releases/download/0.9.0-alpha4/redis-desktop-manager_0.9.0.17_amd64.deb

Устанавливаем его:

dpkg -i redis-desktop-manager_0.9.0.17_amd64.deb

Если при установке появляется ошибка зависимости, устраняем ее с помощью:

apt-get -f install

После этого можно запустить RDM из консоли:

redis-desktop-manager

или прямо из главного меню.
