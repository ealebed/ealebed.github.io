Установка subversion 1.8 в Debian Wheezy

Oct 12, 2015 17:31 · 251 words · 2 minute read svn debian

В репозитории Debian Wheezy доступен пакет subversion версии 1.6 (просмотреть детали). В этой статье давайте разберемся с установкой subversion более свежей версии — 1.8 от WANdisco.

Для начала добавим ключ репозитория (WANdisco GPG key):

wget -q http://opensource.wandisco.com/wandisco-debian.gpg -O- | apt-key add -

Затем добавим сам репозиторий, создав новый файл /etc/apt/sources.list.d/subversion.list:

echo "deb http://staging.opensource.wandisco.com/debian wheezy svn18" > /etc/apt/sources.list.d/subversion.list

Примечание. Конечно, можно добавить репозиторий и в файл /etc/apt/sources.list — это дело вкуса.

Теперь следует обновить информацию о пакетах:

apt-get update

Примечание. Вместо apt-get вы можете использовать aptitude.

Установка SVN (subversion) 1.8 в Debian Wheezy выполняется следующей командой:

apt-get install subversion

Проверить установленную версию пакета subversion можно так:

svn --version
