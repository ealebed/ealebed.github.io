Установка Percona Toolkit в Debian Wheezy

Jan 14, 2016 17:25 · 279 words · 2 minute read percona percona-toolkit debian

Percona-toolkit — это набор специальных утилит для командной строки, разработанных сотрудниками Percona и использующихся для оптимизации производительности баз данных MySQL. Давайте разберемся с установкой percona-toolkit в ОС Debian Wheezy!

Скачиваем .deb-пакет (в наличии также .rpm-пакет и .gz-архив) с официального сайта:

wget percona.com/get/percona-toolkit.deb

Примечание. Если нужна конкретная утилита, то ее можно скачать отдельно, вместо закачки всего набора утилит — используйте wget percona.com/get/TOOL.

Устанавливаем скачанный пакет с помощью пакетного менеджера dpkg:

dpkg -i percona-toolkit.deb

При получении следующей ошибки:

dpkg: dependency problems prevent configuration of percona-toolkit:
 percona-toolkit depends on libio-socket-ssl-perl; however:
  Package libio-socket-ssl-perl is not installed.

устанавливаем отсутствующий пакет libio-socket-ssl-perl:

aptitude install libio-socket-ssl-perl
The following NEW packages will be installed:
  libio-socket-ssl-perl libnet-libidn-perl{a} libnet-ssleay-perl{a}
The following partially installed packages will be configured:
  percona-toolkit

Убедиться, что набор утилит percona-toolkit установлен, можно так:

# dpkg -l | grep percona-toolkit
ii  percona-toolkit                    2.2.16-1                      all          Advanced MySQL and system command-line tools

На этом все, установка percona-toolkit в операционной системе Debian Wheezy закончена, в следующей статье подробно рассмотрим использование утилиты pt-online-schema-change из данного набора.
